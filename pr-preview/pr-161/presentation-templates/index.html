Reveal.js HTML Template

ZIP archive

A single HTML file with no build step. The ZIP bundles all assets so it works locally, on offon.dev, or on any other web host. Preview the live version on offon.dev before downloading.

  • –Agenda table

  • –Speaker cards
  • –Bullet rows with tech tags
  • –Asymmetric split with image
  • –Two-column cards
  • –Contribute cards
  • –Board members
  • –Final / join slide

How to Create a New Deck

  • 1.Download and unzip the template. Rename deck-template.html to your talk name (e.g. my-talk.html).
  • 2.Replace TITLE_PLACEHOLDER in the <title> tag with your event or talk name.
  • 3.Edit the slide sections directly in the HTML. Each <section> is one slide. Delete layouts you don't need.
  • 4.Open the HTML file in any browser for local use. All assets are bundled in the ZIP, so no server is needed.
  • 5.To publish: upload the HTML and asset folders to any web host, or place the file in offon.dev's public/ folder and push to main.
