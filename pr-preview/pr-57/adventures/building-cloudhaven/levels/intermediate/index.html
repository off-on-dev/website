Walkthrough

  1. 1

    Prefer working locally? Clone the repo and open it in any editor that supports the Dev Containers specification (VS Code, JetBrains IDEs, and others). The devcontainer config will be detected automatically.

  2. 2

    Wait ~2 minutes for the environment to initialize.

  3. 3

    Open the Ports tab, find the GCP API Mock at port 30104. Use it to explore mock cloud resources.

  4. 4

    All files are in adventures/02-building-cloudhaven/intermediate/. Run make test to see which tests fail. The tests define the expected behaviour: fix the implementation in modules/district/ to match them. Complete the integration test in tests/integration.tftest.hcl.

  5. 5

    Run make test (all tests should pass) then make apply.

  6. 6

    Run the smoke test: ./smoke-test.sh

  7. 7

    Commit and push your changes to your fork, then trigger the verification workflow on GitHub Actions to officially claim completion.
