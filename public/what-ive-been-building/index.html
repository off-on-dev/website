workstreams 5 · 6 · 7

Digest, FAQ, and AI Skills

5 · Weekly Digest

Open source news roundup, researched, written, and published every Wednesday.

Automated with an AI skill covering research, drafting, and the image prompt. A history file ensures nothing repeats.

Running unbroken since 6 May. The main reason Community Voices is now the largest category at 48 topics.

6 · OSR FAQ

Exported historic requests from Asana and Slack and turned them into an AI skill and FAQ document.

Any incoming request can be answered from it directly, without digging through old tasks. Updated regularly as new types arrive.

Answering a repeat request no longer depends on someone remembering where it was handled last time.

7 · AI Skills Repo

Three skills in the repo: digest, leaderboard, state of community.

Each reads community details from one shared config file — a rename updates all of them at once. Anyone on the team can run them.

Monthly report and weekly digest were both manual before. They now take a fraction of the time.
